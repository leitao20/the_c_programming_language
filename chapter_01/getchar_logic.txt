you can see that getchar is a function that assigns
a character to a variable, as of

int main () {
    int c;
    c = getchar();

    while (c != EOF) {
        putchar();
        c = getchar();
    }
    return 0;
}

you can realize some things, the most important is the
order of the c = getchar() statement. At the right side
a command assigning a value to the left side -a variable.

the syntax is important using just as using comma in English.
We can se that an expression as c = getchar() != EOF will
first operate getchar != EOF, unless we use our "comma";
the parenthesis:

(c = getchar()) != EOF
